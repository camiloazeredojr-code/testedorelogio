# testedorelogio
O Teste do Relógio é um exame simples utilizado no rastreamento de alterações cognitivas e demência. Avalia memória, atenção, planejamento, organização visuoespacial e funções executivas. O paciente deve desenhar um relógio com números e marcar um horário específico. Alterações podem sugerir comprometimento cognitivo.
